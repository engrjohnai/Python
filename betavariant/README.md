📖 Overview

The Beta distribution is a family of continuous probability distributions defined on the interval [0,1]. It is a versatile tool in statistics, often used to model the distribution of probabilities or proportions.

This script specifically implements a scenario where α<1 and β<1, which typically results in a "U-shaped" distribution where values gravitate toward the extremes (0 or 1).
Parameters used:

    Alpha (α): 0.9

    Beta (β): 0.1

    Random Seed: 3 (Ensures reproducibility)
