Write a function called equation_writing() that takes three parameters: a, b, and c. 
The function should print an equation in the format a x + b = c, where a, b, and c are the values passed as arguments. 
The x and other symbols in the equation remain constant.

Note:

    Do not handle user input within this function.

    Do not call the function; only implement it.

    The function should work for any valid numeric values of a, b, and c.

    The function should print the equation, not return it.

Sample Input 1:

3 4 3

Sample Output 1:

3 x + 4 = 3
